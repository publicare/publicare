Objetos Vencidos

Armazena os itens que são expirados automaticamente pelo sistema. Os objetos com data de validade vencida são listados em Objetos Vencidos.

Na tela Objetos vencidos é possível editar o objeto expirado ou apagá-lo em definitivo.


   Como visualizar os objetos vencidos?

Se dirigir a área do site em que possui permissão de editor.
Na parte superior da página clicar em Opções de Menu.
Em seguida, clicar em Objetos Vencidos
Serão listados todos objetos vencidos da área.
  • Para apagar os objetos vencidos:
    Marcar os itens desejados (1) e clicar em Apagar Selecionados (3);
  • Para editar o objeto vencido:
    Clicar sobre o ícone Editar Objeto (4) referente ao item desejado;
  • Utilizar o botão Inverter Seleção (2) para auxiliá-lo a selecionar os ítens;
  • Para sair da tela, clicar no ícone Visualizar Objeto



Importante
• Somente os usuários administrador e editor possuem permissão para visualizar a opção Objetos Vencidos.

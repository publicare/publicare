Visão Geral Sobre Contas de Usuários

Uma conta de usuário define as ações que um usuário pode executar no site.

Uma conta de usuárioConta de Usuário

Registro que consiste em todas as informações que definem um usuário para o Publicare. Inclui o nome de usuário e a senha necessários para que o usuário faça logon e os direitos e permissões que o usuário tem para utilizar os recursos do Publicare. estabelece os privilégios atribuídos a cada usuário

Há quatro tipos de conta de usuário disponíveis no Publicare:

  • AdministradorAdministrador

    Usuário que gerencia o site tendo acesso a todos os recursos do Publicare. O administrador realiza alterações gerais nas páginas do site, além de poder criar, alterar e excluir as contas de outros usuários.         ;
  • EditorEditor

    Usuário que possui acesso limitado aos recursos do publicare. Esse usuário só pode manipular os objetos do site nas áreas que possuir permissão. Pode possuir contas de nível mais baixo subordinadas aos seus cuidados.    ;
  • AutorAutor

    Usuário que só possui permissão para colocar conteúdo no site. Este tipo de usuário é subordinado a um usuário de nível mais alto por não ter acesso a determinados recursos do Publicare.    ;
  • RestritoRestrito

    Usuário que só possui permissão para navegar no site, podendo visualizar objetos não publicados.    .
   Conta de Administrador do Site

Pessoa responsável pelo gerenciamento do site e contas de usuários, atribuindo senhas e permissões e ajudando usuários com problemas no site. Esse usuário:

  •   Pode criar e excluir contas de outros usuários.
  •   Pode alterar senhas e tipos de contas dos outros usuários.
  •   Pode realizar alterações gerais nas páginas do site.
  •   Pode apagar em definitivo os objetos excluídos por outros usuários.
  •   Pode criar classes e alterar suas propriedades.

   Conta de Editor

A conta de editor destina-se aos usuários que não tem permissão para manipular determinados recursos do Publicare. Um usuário com este tipo de conta:

  •   Pode alterar seus dados pessoais, tais como e-mail e senha de logon.
  •   Pode restaurar objetos excluídos.
  •   Não pode apagar em definitivo os objetos excluídos do site.
  •   Pode possuir contas de nível mais baixo subordinadas aos seus cuidados.
  •   Pode criar, editar, apagar, publicar, despublicar e rejeitar objetos nas áreas do site que possuir permissão.
  •   Não pode alterar o tipo de sua própria conta. Um usuário com uma conta de administrador do site deve fazer esse tipo de alteração.

   Conta de Autor

A conta de autor destina-se aos usuários que não têm permissão para manipular a maioria dos recursos do Publicare. Um usuário com este tipo de conta:

  •   É subordinado a um usuário de nível mais alto.
  •   Pode alterar seus dados pessoais, tais como e-mail e senha de logon.
  •   Não pode recuperar objetos apagados. Sendo necessário entrar em contato com o usuário cadastrado no publicare como seu supervisor.
  •   Não pode publicar objetos, sendo estes publicados pelo usuário cadastrado no publicare como seu supervisor.

   Conta Restrito

A conta restrito destina-se aos usuários que não possuem permissão para colocar conteúdo no site. Um usuário que tenha feito logon nesse tipo de conta:

  •   Não pode colocar conteúdo no site.
  •   Pode navegar pelo site e ver objetos não publicados.
  •   Pode alterar seus dados pessoais, tais como e-mail e senha de logon.
  •   Não pode alterar o seu tipo de conta.

Teclas de atalho

Você pode usar teclas de atalho para editar, criar, apagar, publicar, rejeitar publicação, listar conteúdo, visualizar índice de objetos e efetuar LOGOFF.

Para Pressione
Editar objeto atual ctrl + alt + shift + E
Criar objeto dentro do objeto atual ctrl + alt + shift + C
Apagar objeto atual ctrl + alt + shift + D
Publicar objeto atual ctrl + alt + shift + P
Rejeitar publicação do objeto atual ctrl + alt + shift + R
Lista contéudo, no objeto atual ctrl + alt + shift + L
Visualizar índice do objeto ctrl + alt + shift + V
Efetuar LOGOFF no sistema ctrl + alt + shift + O


Observações:
• Alguns dos itens podem não funcionar, caso você não tenha permissão dentro do objeto.
• Devido a diferentes browsers disponíveis no mercado, sua tecla de atalho pode funcionar utilizando somente: ctrl + alt + LETRA

Você também pode acessar o histórico das teclas de atalho clicando sobre a aba Dicas, disponível na parte superior das páginas somente quando você estiver logado.

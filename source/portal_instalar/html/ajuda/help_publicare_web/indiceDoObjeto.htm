Índice do Objeto - Como Acessar?

Ao acessar o Índice do Objeto, é gerada a tela Preview de Objeto onde são exibidas as propriedades do objeto selecionado. É através do índice do objeto que acessamos também a guia Workflow e Log Status

   Procedimento passo a passo

Localizar a página desejada para visualização do Índice do Objeto.
Como exemplo, usaremos uma página qualquer do site.
No topo da página, clicar na aba Opções de Menu
No meu lateral esquerdo, clicar na opção Índice do Objeto
A tela abaixo será apresentada disponibilizando três guias
para consulta dos dados referentes ao objeto selecionado
  •  Índice do Objeto (1);
  •  Workflow (2);
  •  Log Status (3).

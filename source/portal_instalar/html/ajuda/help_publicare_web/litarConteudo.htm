Listar Conteúdo

A opção Listar Conteúdo é utilizada para listar os objetos contidos em uma página, os seus filhos e propriedades.

  1. O primeiro passo é se dirigir à página desejada.
  2. No topo da página, clicar em Opções de Menu > Listar Conteúdo ou pressione ctrl + alt + shift + L, para listar os objetos da página.
  3. Para realizar as operações referentes aos botões copiar para pilha, publicar, solicitar, despublicar e apagar itens), a caixa de seleção dos objetos que deseja manipular devem estar marcadas .

   Siga os passos ilustrados nas figuras

Localizar no site a página desejada
Na parte superior da página, clicar na aba Opções de Menu
Será aberto o menu de administração de objetos. Clicar na opção Lista Conteúdo
Serão listados os objetos da página.
Para realizar as operações dos botões, é necessário assinalar os objetos desejados

  • (1) Botão que inverte a seleção dos objetos;
  • (2) Botão que publica os objetos selecionados;
  • (3) Botão que despublica os objetos selecionados;
  • (4) Botão que apaga os objetos selecionados;
  • (5) Botão que duplica os objetos selecionados;
  • (6) Botão que copia os objetos selecionados para a área de transferência (Pilha), podendo colá-los no local que desejar e possuir permissão;
  • (7) Pilha: ferramenta de cópia/cola de objetos.



Observação:
• Ao listar o conteúdo de uma página, caso não esteja disponível a caixa de seleção do objeto, significa que o usuário logado não possui permissão para manipulá-lo.

Log de Acesso

Log de acesso é um recurso do Publicare que permite ao usuário logado obter o número de vezes que uma determinada página do site é visitada. É um dos instrumentos mais importantes para medir a audiência de um site ou portal.

O usuário poderá visualizar o número de acessos por dia, mês e ano.


   Como verificar o número de acessos a uma página?

Abrir a página desejada para visualização dos acessos.
No topo da página clicar em Opções de Menu.
Ao abrir o menu de administração de objetos,
clicar na opção Log de Acesso
O quadro Log de Acessos será aberto.

  • No quadro (1) é possível verificar a hora e o número de acessos clicando sobre o dia desejado;
  • No quadro (2) é possível verificar o número de acessos correspondentes aos dias do mês selecionado;
  • No quadro (3) é possível verificar o número de acessos correspondentes aos meses do ano selecionado;
  • No quadro (4) é possível filtrar a categoria de acessos desmarcando os tópicos desejados e clicando em Executar. Assim, no resultado serão exibidas somente as opções marcadas.

Para sair da tela, clicar em Visualizar Objeto

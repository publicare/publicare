Aba Avançado

A aba Avançado é acessada através do formulário do objeto. Ao ser acessada é possível fazer um comentário a respeito do objeto, alterar o script de exibição, o peso e o dono do objeto.


  • (1) Descrever o objeto caso necessário;
  • (2) A troca de pele permite que se tenha diferentes padrões de visualização (layout) para cada pele;
  • (3) O script de exibição é alterado somente quando for necessário visualizar um objeto de modo diferente do padrão utilizado para os objetos da mesma classe. Caso contrário permanecer a opção "seleção automática";
  • (4) Todo membro que cria um conteúdo dentro do Publicare, se torna o dono do seu conteúdo. O dono de um objeto pode ser alterado;
  • (5) Utilizar no campo peso valores numéricos para determinar a ordem dos objetos na página.


Observação
• A troca de pele só é realizada por um usuário Administrador.
• Quando é alterada a pele de um objeto pai automaticamente é alterada a pele dos objetos filhos.
• O usuário autor não tem permissão para alterar a opção Dono do objeto.
• O usuário editor só poderá alterar a opção Dono do objeto em objetos que ele criou, sendo possível colocar como dono do objeto somente os seus subordinados. Caso contrário, entrar em contato com o Administrador do site.

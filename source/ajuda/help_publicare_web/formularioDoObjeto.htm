Formulário do Objeto

No publicare, as classes são usadas para criar objetos. Quando definimos uma classe de objetos, estamos na verdade definindo quais propriedades o objeto herdará. Assim, é gerado o formulário do objeto onde as propriedades da classe são apresentadas em forma de campos que devem ser preenchidos pelo usuário para criar o objeto.

Esse é o formulário de um objeto do tipo Arquivo

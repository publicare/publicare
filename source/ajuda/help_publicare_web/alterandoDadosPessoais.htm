Alterando Dados Pessoais

Use este procedimento para alterar o seu e-mail, ramal e a senha usada para fazer logon no Publicare.

  1. Na parte superior da página clicar na aba Opções de Menu



  2. No painel de administração de Objetos clicar na opção Dados Pessoais



  3. Alterar os dados desejados e clicar em Gravar

Opções de Gravação de Objetos

Ao abrir o formulário de criação ou edição de um objeto, são disponibilizados diferentes tipos de botões para gravação dos dados dispostos no formulário.

   Botão Gravar
  • O botão Gravar é utilizado quando o usuário deseja apenas gravar os dados informados nos campos do formulário de edição ou criação do objeto.
  • O objeto é salvo e só pode ser visualizado pelo usuário que estiver logado, pois ainda não se encontra publicado.
  • Para que o usuário final possa ver o objeto criado é necessário publicá-lo.

   Botão Gravar e Publicar
  • O botão Gravar & Publicar é utilizado quando o usuário deseja gravar os dados informados nos campos do formulário de edição ou criação do objeto e ao mesmo tempo publicá-lo.
  • O objeto é salvo e disponibilizado para que o usuário não logado possa vê-lo.

   Botão Gravar e Solicitar
  • O botão Gravar & Solicitar é utilizado quando o usuário deseja não somente gravar os dados informados nos campos do formulário de edição ou criação do objeto, como também solicitar a sua publicação.
  • O usuário solicita a publicação do objeto pois não possui permissão para publicá-lo.
  • O responsável por publicar os objetos desse usuário receberá por e-mail a solicitação de publicação. Este, poderá publicá-lo ou não.
  • Até que o objeto seja publicado, o usuário não logado não poderá vê-lo.

   Botão Gravar e Inserir Outro
  • O botão Gravar & Inserir Outro é utilizado quando o usuário deseja salvar os dados do formulário de edição ou criação do objeto e logo após criar um novo objeto do mesmo tipo (ex: Diretório, Arquivo, Link etc).
  • Após a gravação, é aberto um novo formulário para criação de um novo objeto.
  • Os objetos criados, só podem ser visualizados pelos usuários que estiverem logado, pois ainda não se encontram publicados. Assim, é necessário publicá-los ou solicitar a publicação dos objetos.

Observação:
• As opções de botões mudam de acordo com a permissão dada ao usuário.

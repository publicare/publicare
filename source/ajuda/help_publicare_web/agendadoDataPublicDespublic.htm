Agendando data para publicação / despublicação do objeto

No formulário do objeto exitem campos que lhe permite especificar a data e a hora em que o documento será publicado e despublicado automaticamente.

  •  Data de Publicação: permite você especificar uma data de efetivação para este conteúdo. Até esta data chegar, o conteúdo não será publicado. Até que esta data ocorra, o conteúdo não será publicado.
  •  Data de Validade: permite você especificar uma data para este conteúdo expirar. Despois desta data o conteúdo não estará mais publicado. Se esta data não for especificada, então não terá efeito caso o objeto venha a ser publicado.

Os campos Data de Publicação e Validade são acompanhados do campo hora.
